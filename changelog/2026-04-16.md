## Architecture Decision

### Problem
The gRPC PHP extension (`grpc.so`) fails to load on PHP 8.3/8.4 Alpine Linux due to ABI compatibility issues.

### Solution
HTTP/REST bridge architecture - Laravel uses HTTP client instead of gRPC stubs.

```
┌─────────────┐    HTTP REST    ┌──────────────────┐
│   Laravel   │ ──────────────▶ │  Python FastAPI  │
│  (PHP 8.3)  │                 │  (Port 8001)     │
└─────────────┘                 └──────────────────┘
```

### Endpoint Mapping
| Laravel | Python REST | Purpose |
|---------|-------------|---------|
| `POST /api/ai/analyze/{diagram}` | `POST /api/analyze/text` | Diagram suggestions |
| `GET /api/ai/health` | `GET /health` | Health check |
